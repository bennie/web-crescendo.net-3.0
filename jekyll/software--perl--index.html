Project Summary:

Under the username of BENNIE I have several modules published and available on CPAN. Full details can be found in my account on CPAN, but here is a brief summary:

  • Text::TabFile v1.10 (Aug 2, 2006)

    A module for dynamic iteritive parsing of tab delimited files.

  • Text::Delimited v2.00 (Jul 31, 2006)

    A module for dynamic iteritive parsing of character delimited files.

  • Term::Activity v1.06 (Sep 10, 2004) - registered

    A module used for terminal display of non-bounded processing. Speed aware to keep from flooding the terminal, but still show activity.

  • Acme::Umlautify v1.01 (Apr 7, 2004) - registered

    This module, part of the humor Acme namespace, will add Umlauts to about anything it can.

  • Cache::Bounded v1.03 (Apr 6, 2004) - registered

    A size aware cache that is designed for speed of write and minimal measurement overheads

  • Rcs::Parser v0.02 (Apr 2, 2004)

    Directly parse and read RCS files without needing a binary version of RCS.
