Project Summary:

The User Ignore Mod is an add-on for the very popular phpBB. It is a mod that allows users to ignore the post of other users.

Originally this module was put together by GunnerX back in 2002. I took it over to bring it up to being compatible with phpBB v2.0.6-2.0.10 security and more friendly with the urrent version of EasyMod. I also added an administrative page for ignore statistics.
