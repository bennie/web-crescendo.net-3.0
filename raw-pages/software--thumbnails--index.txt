<font size="5"><p>Thumbnails (v 1.0.0)</p></font>

<p>thumbnails is a perl program that generates a set of thumbnails and
core HTML to visually index a directory of images.</p>

<p>It can take a <a href="example/images/">very large directory of
images</a> and create <a href="example/images/images.html">block of HTML
and thumbnail images</a> to match it.</p>

<p>This program is released under the <a
href="../gnu.html">GNU Public license</a> and comes with some
<a href="thumbnails-1.0.0/INSTALL">basic documentation</a>.</p>

href="thumbnails.1.0.0.tar.gz">thumbnails.1.0.0.tar.gz (9.4k)</a> ]</b></p>
