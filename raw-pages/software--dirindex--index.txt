<p align="center"><font size="5">dirindex</font></center>
 
<p><b>Summary:</b></p>

<p>Dirindex is a simple alteration of the <a href="../thumbnails">thumbnails</a>
script. It is designed to reduce processing and synchronize a changing set 
of directories. It can be run regularly from a cron job with command 
parameters.</p>

<p>This script is currently in beta. More documentation to follow.</p>
